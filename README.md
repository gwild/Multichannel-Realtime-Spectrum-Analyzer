This code implements a multichannel real-time spectrum analyzer in Rust. main.rs handles user input for device and configuration selection. Main.rs then starts a thread for audio_stream.rs which samples data from portaudio and pushes it into a circular buffer continuously. Main.rs also starts a thread for fft_analysis.rs which continuously processes clones of the circular buffer, computing the Fast Fourier Transform (FFT) and partials results. Ftt_analysis calls plot.rs which visualizes the results in an interactive GUI, updating continuously. The GUI has various elements to make adjustments in real-time for various plot elements and circular buffer resizing.  Since the code uses portaudio for the audio backend, it will run on Linux and OSX, but not Windows since portaudio is not available for Windows (though you may be able to build it yourself).
